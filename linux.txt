文件类型
7种文件类型：
- regular 普通文件
d directory 目录文件
c charactor 字符设备文件
b block 块设备文件
l link 符号链接文件
p pipe 管道文件
s socket 套接字文件
文件权限rwxr-xr-x
u 拥有者
g 所属组
o 其他用户
a 所有
修改文件权限
<1>chmod u/g/o/a +或者- r/w/x filepath
<2>chmod 加上所需要的八位制数（666）filepath
修改文件的所属用户和所属组
chown user:group filepath

文件的硬链接：相当于复制一个文件 但是名字不同
（一点不懂） inode是同一个分区文件的唯一标识
硬链接文件不能跨分区
创建硬链接
ln srcpath dest
cp -l srcpath dest
软连接：相当于快捷方式
ln -s srcpath dest
cp -s srcpath dest

    find（便利磁盘空间）
        find path options
        options
            -name 按名字查找
            find  /etc  -name  "passwd"
            -size 按文件大小查找
            find  /etc/  -size  n 
            -mtime  修改时间
            find  /etc/  -mtime  n(天)
            -type (类型)
            find  /etc/  -type  f/d/c/b/p/l/s
            -exec  对于查找到的结果执行
            find  /etc/  -name  "hello"  -exec  ls  -l  {}  \;  (直接执行)
            -ok
            find  /etc/  -name  "hello"  -ok  ls  -l  {}  \; (执行前询问)
            -print  打印查找结果
            find  /etc/  -name  "hello"  -print  -exec  ls  -l  {}  \; 
            -a  连接多个查找选项
            find  /etc/  -type  d  -a  -size  +18k

        yum  install  xxxx

（完全不会） 本地yum源配置:
1.将镜像文件挂载到文件系统
2.创建镜像的挂载点 mkdir /mnt/centos7
3.光盘镜像在/dev/cdrom目录
4.挂载光盘镜像
mount -t iso9660 -o loop /dev/cdrom /mnt/centos7
5.配置镜像的yum源
vim /etc/yum.repos.d/my.repo

            [local_centos7]
            name=centos7
            baseurl=file:///mnt/centos7
            enabled=1
            gpgcheck=0

    已下载的rpm软件包安装：
        xxx.rpm
        rpm -ivh xxxx 安装已下载的rpm包
        rpm -e xxxx 卸载
        rpm -U xxxx 更新
        rpm -qa 查询系统中所有已安装的rpm包
        rpm -qf /usr/bin/ls 查询ls对应的软件包

vim 编辑器的使用：
vim filepath --->打开 --->一般模式
--->进入编辑模式编辑(iIaAoOsS)
--->从编辑模式进入到一般模式(Esc / Ctrl+c)
--->从一般模式进入到命令行模式(:w 保存 ：q 退出)
一般模式：
2yy 	复制两行
p   	粘贴
2dd 	剪切两行
u   	撤销
Ctrl+r 	还原
x/X 	删除光标字符
r		替换
R 		一直替换到Esc为止
块选择：
Ctrl+v 	方向键选择
gg 		首行
GG 		末行
5G 		第五行
命令行模式：
w 		保存
q 		退出
q！		强制退出
w filpath 	另存为
%s /old/new/g 将所有的old替换为new
2,5s/old/new/g 将第二行到第五行的old 替换为new
set nonumber  取消行号
set nu 	 显示行号
set ts=4 将Tab设置为4个字符宽度
